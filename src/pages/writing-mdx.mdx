# Writing With MDX

Author docs pages in `.mdx` and add a route entry in `src/nav.ts`.

## Add a New Page

1. Create `src/pages/your-page.mdx`.
2. Import it in `src/App.tsx`.
3. Add a nav item in `src/nav.ts` and a route mapping in `src/App.tsx`.

Inline code like `const value = 42` is already styled for readability on glass surfaces.
